# war-challenge-computer-vision
Computer Vision system to get useful information about the fied in WAR Online

## Installation

### Project
```bash
# Python 3.10+
poetry install
poetry shell
```

### Tesseract
```bash
sudo apt install tesseract-ocr
sudo apt install libtesseract-dev
```
